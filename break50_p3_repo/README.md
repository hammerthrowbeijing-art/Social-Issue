# Break 50 P3 Repository

This repository is organized to satisfy the course requirement for a public GitHub submission: it includes the project plan, reproducible code, documentation, and analysis outputs, but it does **not** include the raw `B50_X_COMMENT.xlsx` dataset.

## Repository Contents

- [`docs/P3_plan.md`](./docs/P3_plan.md): project plan for the Break 50 study
- [`docs/analysis_report.md`](./docs/analysis_report.md): formal write-up of the current analysis output
- [`docs/final_report.md`](./docs/final_report.md): fuller narrative report that integrates the methods, findings, and interpretation
- [`scripts/run_analysis.py`](./scripts/run_analysis.py): reproducible analysis pipeline
- [`outputs/figures/`](./outputs/figures/): figures generated by the script
- [`outputs/tables/`](./outputs/tables/): markdown tables and machine-readable analysis summary

## Research Question

The project asks whether **author's attention quantity** is associated with five comment-level outcomes in the Break 50 dataset:

- `retweets count`
- `likes`
- `Comment word length`
- `Comment views`
- `followers`

## How To Reproduce

Keep the Excel file local and outside the repository, then point the script to it with an environment variable:

```powershell
$env:BREAK50_DATA_PATH="C:\Users\Larry.Nie\Downloads\B50_X_COMMENT.xlsx"
py -X utf8 scripts/run_analysis.py
```

If `BREAK50_DATA_PATH` is not set, the script will try the default local path above.

## Notes For GitHub Submission

- Do not upload `xlsx`, `csv`, or other raw data extracts.
- Commit only code, markdown documentation, figures, and meaningful analysis outputs.
- If group members continue revising the analysis, rerun `scripts/run_analysis.py` so the figures and markdown tables stay synchronized with the report.
