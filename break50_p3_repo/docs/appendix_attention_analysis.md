# Appendix: Earlier Author-Attention Analysis

This appendix preserves the earlier exploratory analysis that focused on **author's attention quantity** as the main independent variable. The instructor feedback suggested that this framing was mechanically sound but not yet aligned with the P3 emphasis on contested discourse, so the main report was revised rather than discarded.

## What This Appendix Contains

The earlier workflow examined whether author's attention quantity was associated with:

- `retweets count`
- `likes`
- `Comment word length`
- `Comment views`
- `followers`

Those outputs are still available in the repository as legacy exploratory material:

- [`attention_coefficients.png`](../outputs/figures/attention_coefficients.png)
- [`attention_quartiles.png`](../outputs/figures/attention_quartiles.png)
- [`correlations.md`](../outputs/tables/correlations.md)
- [`model_summary.md`](../outputs/tables/model_summary.md)
- [`quartile_summary.md`](../outputs/tables/quartile_summary.md)

## Why It Is Still Useful

The earlier analysis remains valuable for two reasons.

1. It established a reproducible infrastructure for reading the local data, generating markdown tables, and exporting figures.
2. It showed that author-side attention and audience-position variables still matter as secondary controls even after the project pivots toward discourse coding.

## How It Relates To The Revised Project

In the revised P3 framing, `author's attention quantity` is no longer the main research question. Instead, it is treated as a secondary control variable that helps separate discourse-position effects from account-level visibility advantages.
